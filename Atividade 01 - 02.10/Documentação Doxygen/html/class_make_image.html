bool MakeImage::saveImage ( const std::string &  nomeDoArquivo)

Esta função inicializa o uso da biblioteca ImageMagick e, através de uma de suas funções próprias, converte os arquivos .ppm gerados pelas funções anteriores em imagens .png.

Parâmetros
nomeDoArquivoRepresenta o nome do arquivo de imagem .png que será gerada a partir dos .ppm
Retorna
true Significa que a criação e escrita da imagem foram bem-sucedidas
false Significa que houve uma falha no processo de salvar a imagem a partir do .ppm
